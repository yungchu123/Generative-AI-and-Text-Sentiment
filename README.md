# Generative-AI-and-Text-Sentiment
This project is a Flask-based web application that generates images based on text input and performs sentiment analysis. The app uses various tools and APIs for image generation and sentiment analysis.

## Features
- Image Generation: Uses OpenAI's DALL-E API to generate images based on text prompts.
- Sentiment Analysis: Analyzes the sentiment of the input text using the TextBlob library.
